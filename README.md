# Program Pengolahan Nilai Mahasiswa

## Deskripsi Program
Program ini dibuat untuk memenuhi Ujian Akhir Semester (UAS) mata kuliah
Algoritma dan Pemrograman Lanjutan.

Program berbasis bahasa C ini digunakan untuk mengelola data nilai mahasiswa
meliputi NIM, nilai tugas, UTS, dan UAS menggunakan array dan fungsi rekursif.

---

## Fitur Program
- Input data mahasiswa
- Menampilkan data mahasiswa
- Menambah data mahasiswa
- Mengubah data mahasiswa
- Menghapus data mahasiswa
- Menghitung rata-rata nilai
- Menampilkan nilai tertinggi dan terendah
- Fungsi rekursif (Faktorial dan Fibonacci)
